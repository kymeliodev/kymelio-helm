  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "argocd.fullname" . }} {{ (first .Values.service.ports).port }}:{{ (first .Values.service.ports).port }}

Then open the UI at:

  http://127.0.0.1:{{ (first .Values.service.ports).port }}

The argocd-server requires the argocd-repo-server and a Redis instance to be
running, and the Argo CD CRDs (Application, ApplicationSet, AppProject) must be
installed separately before creating Applications.

Run the bundled connection test with:

  helm test {{ .Release.Name }} --namespace {{ .Release.Namespace }}
