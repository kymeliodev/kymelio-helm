This release expects an external PostgreSQL database:

  PostgreSQL: {{ .Values.externalDatabase.host }}:{{ .Values.externalDatabase.port }}

Make sure the database password is provided through extraEnv as
ZITADEL_DATABASE_POSTGRES_USER_PASSWORD before the server can start. TLS is
disabled in the pod, so place ZITADEL behind a TLS terminating ingress.

Run the bundled connection test with:

  helm test {{ .Release.Name }} --namespace {{ .Release.Namespace }}
