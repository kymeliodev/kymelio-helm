Port forward the web UI to your workstation:

  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "gitlab.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.port }}

Retrieve the initial root password (valid for 24 hours after first boot):

  kubectl --namespace {{ .Release.Namespace }} exec -it sts/{{ include "gitlab.fullname" . }} -- \
    cat /etc/gitlab/initial_root_password

Set external_url through the omnibusConfig value so links resolve correctly.
