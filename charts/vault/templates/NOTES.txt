Reach the API from inside the cluster at:

  {{ include "vault.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.port }}

Port forward to your workstation:

  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "vault.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.port }}

Then set the client address and check status:

  export VAULT_ADDR=http://127.0.0.1:{{ .Values.service.port }}
  vault status
